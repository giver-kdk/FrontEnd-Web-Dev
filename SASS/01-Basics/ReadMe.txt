- TO start, install 'Live SASS Compiler' extension by 'Glenn Marks' for Vs-Code
- Configure the 'Live SASS Compiler' in 'settings.json'

- To format SASS and SCSS files, 
	- Install 'SASS Formatter' extension for Vs-Code
	- Install RubyGems package by running: 
		- sudo apt install ruby ruby-dev
	- Install SASS commandline tool using : 
		- sudo gem install sass
